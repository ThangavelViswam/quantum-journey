## Chapter 2: Linear Combinations, Span & Basis Vectors

### Basis Vectors
- i-hat = [1, 0] → unit vector along x-axis
- j-hat = [0, 1] → unit vector along y-axis
- Any vector [3, -2] = 3(i-hat) + (-2)(j-hat)

### Key Concepts
- Linear combination = scaling two vectors and adding: a·v + b·w
- Span = all vectors reachable by varying the scalars
  - Two non-parallel vectors → entire 2D plane
  - Two parallel vectors → just a line
- Linearly dependent = one vector is redundant (already in span)
- Linearly independent = each vector adds a new direction

### Quantum Computing Connection
- Qubits use basis states |0⟩ and |1⟩ — exact analogy to i-hat and j-hat
